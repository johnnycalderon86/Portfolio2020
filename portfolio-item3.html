After working with both React Native and Flutter I wanted to have a go at a native Android application. I decided to use Kotlin as a language instead of Java. Mainly because I had read it was easier to learn, and also because Google had announced that Kotlin would be the preferred language going forward for Android development. Using Android Studio was also a good experience. Being able to alternate between the layout XML tab and the design tab is a nice feature.
